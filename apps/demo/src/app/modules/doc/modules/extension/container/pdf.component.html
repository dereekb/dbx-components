Drop PDFs (application/pdf) and/or images (PNG/JPEG) into the upload area. Drag the handle on each row to reorder. Pages are appended in list order; image entries become a single page sized to the image.

Each slot owns its own upload area and inline file display, and is tagged with a `slotId`. The validator directive aggregates each slot's readiness — Preview/Download stay disabled until every required slot has a ready file. Optional slots (here, Insurance Card) only block when they contain an invalid file. State-aware icons projected through the slot's header use the `dbxPdfMergeEditorFileUploadHasState` directive.

Slot capacity defaults to `1` for single-file slots and to `maxFiles` when set. Once the slot reaches capacity its `dbx-file-upload` is hidden — the user must remove an existing entry before adding more.

Configure the editor with `imageCompression` to downscale and (optionally) convert images on upload, and `outputSizeLimits` to surface warn/error banners. Crossing `errorBytes` blocks Preview/Download via the store's validity gate. Each compressed entry shows a "Resized" / "Converted PNG → JPEG" chip with the original byte count.

Wrap the trigger in [dbxPdfMergeEditorStore] and (optionally) a dbxAction with a handler. The dbxPdfMergeUploadButton directive registers a DbxButtonInterceptor on the host dbx-button — when the dialog returns a confirmed Blob it calls readyValue(blob) on the surrounding action source, and the sibling dbxActionButton directive handles the resulting click and pipes working/disabled state back to the button. Close the dialog without confirming to keep the editor entries for a later attempt.

Pass a DbxPdfMergeUploadButtonConfig whose customDialogContent renders any custom dbx-pdf-merge-editor configuration (slots, multiple uploaders, etc.) and whose uploadButtonConfig overrides the dialog footer Upload button (text, icon, color, raised, flat/stroked/tonal/basic variant flags). Leave the inner component config's injector unset so the injected component inherits the dialog's injector and sees the same store.
