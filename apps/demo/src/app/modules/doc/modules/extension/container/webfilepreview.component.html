Content can be registered using the DbxWebFilePreviewService's registerEntry function. The service will then handle previewing the content based on the provided configuration.

Entries can be configured with a preview component function and an optional preview dialog function, which can be used to preview the content in a dialog.

Entries can also be configured via the DBX_WEB_FILE_PREVIEW_SERVICE_ENTRIES_TOKEN injection token.
