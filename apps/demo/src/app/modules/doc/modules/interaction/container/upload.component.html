The dbx-file-upload component makes it easy to add file upload functionlity to a component.

It supports drag/drop to upload, is pre-styled, and has several configurations, including multiple and accept .

It can be configured like a usual input field, with both the multiple and accept attributes.

accept

The accept attribute can be the traditional string you would typically pass to the accept attribute on an input, an array of strings which will get concatinated together, or a FileAcceptFunction function.

The values will be pre-separated in the returned event.
