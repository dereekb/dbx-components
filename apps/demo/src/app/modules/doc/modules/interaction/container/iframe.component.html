dbx-iframe handles creating an iframe then replacing the content with the content from a url or a SafeResourceUrl.

The allow attribute is set to "autoplay". If you need to change the allow attributes, extend DbxIframeComponent and replace the template.

Configuration

Scrolling

dbx-iframe can be configured to scroll the iframe or not.

Sanitize URL

dbx-iframe can be configured to sanitize the url or not.

dbx-embed handles creating an embed element then replacing the content with the content from a url or a SafeResourceUrl.

This can be useful for displaying a PDF or other arbitrary media types.
