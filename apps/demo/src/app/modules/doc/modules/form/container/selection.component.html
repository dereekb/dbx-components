Select

The typical select fields.

Source Select

Selection field that provides a button for finding/retrieving additional values through an arbitrary source. The selectable value must always be a key. Several delegate functions handle retrieving values from pre-configured sources, and metadata via the existing values.

Dbx List

Pickable

Pickable fields are use to display a list of preset/static values that the user can choose from.

Searchable

Seaarchable fields are more complex fields that support searching for items that the user can pick.

The intention of this field is to make searching and picking a single item easy, and leaves a lot of power to the developer.

This field is highly customizable, allowing for customization on a per-pick basis.

Anchors

It can also be configured to display content within a dbx-anchor value, allowing for navigation as a result of searching.
