Default avatar

This avatar has no url or other contextual info set, so it shows just the icon.

Sizes can be set using the avatarSize input, or adding the "dbx-avatar-large" or "dbx-avatar-small" classes.

Avatar with url

This avatar has a url set, and shows the image from the url.

Custom icon

This avatar has no url/context set, but does have a custom icon set.

Avatar with broken url

This avatar has a broken url set. The error will be detected and the fallback icon will be displayed.

The second avatar has a custom "cancel" icon set.

Named avatars

Each name maps to a curated color from the curated color set. The same name always maps to the same color (deterministic).

Initials use the first letter of the first two words (e.g. "Michelle B" → "MB").

@for (name of demoAvatarNames; track name) { }

Sizes and styles

Initials scale with the avatar size, and support both circle and square styles.

Characters

Single or double character strings pass through verbatim (e.g. "A" → "A", "BB" → "BB").

@for (characters of demoAvatarCharacters; track characters) { }

Name with broken url

When an image url fails to load and a name is set, the avatar falls back to the initials (instead of the icon).
