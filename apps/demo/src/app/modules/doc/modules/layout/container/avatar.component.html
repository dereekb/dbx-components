Default avatar

This avatar has no url or other contextual info set, so it shows just the icon.

Sizes can be set using the avatarSize input, or adding the "dbx-avatar-large" or "dbx-avatar-small" classes.

Avatar with url

This avatar has a url set, and shows the image from the url.

Custom icon

This avatar has no url/context set, but does have a custom icon set.

Avatar with broken url

This avatar has a broken url set. The error will be detected and the fallback icon will be displayed.

The second avatar has a custom "cancel" icon set.

DbxAvatarViewServiceConfig

You can provide this configuration in root.app.config.ts to configure the DbxAvatarViewService.
