CSS Custom Properties

  • --dbx-sidenav-width — Full mode width (default: 240px)
  • --dbx-sidenav-mobile-width — Mobile overlay width (default: 80vw)
  • --dbx-sidenav-icon-width — Icon rail width (default: 65px)
  • --dbx-sidenav-background — Drawer background color
  • --dbx-sidenav-text-color — Drawer text color
  • --dbx-sidenav-divider-color — Drawer border/divider color
  • --dbx-sidenav-scrim-color — Backdrop overlay color
  • --dbx-sidenav-active-accent — Active item accent color
  • --dbx-sidenav-active-item-background — Active root item background
  • --dbx-sidenav-active-child-item-background — Active child item background
  • --dbx-sidenav-active-child-item-border-width — Active item border indicator width (default: 2px)

This layout uses displayMode="mobile" with position="end" and a colored pagebar with a menu toggle button. The pagebar height is increased to 73px via --dbx-pagebar-height . The sidenav is always hidden and opens as a right-side overlay when the menu button is clicked.
